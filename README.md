🐶 a bbs project/完整的BBS论坛

使用技术:

* 后台
	* 使用`Spring Boot` 构建整个项目 去除 XML 配置
	* `Maven`构建项目
	* `Jenkins`作为持续集成
	* 采用`SpringCloud`作为RPC框架
* 前台
	* 采用vue.js+elementUI架构
